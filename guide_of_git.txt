high frequency used command for git:
1. git clone
example: git clone https://github.com/weixk/PersonalBlog.git
clone total project from remote repository

2. git pull
example: git pull
check for the updates of the code, and download the new code into the project
usage: run the command just in the folder has .git file. git will pull the code according to the info saved in the .git file

3. git status
example: git status
check for the status of files that git tool managed

4. git push
example: git push
push code to the remote repository

5. git commit
example: git commit -a -m "first commit"
commit the change to the local repository
